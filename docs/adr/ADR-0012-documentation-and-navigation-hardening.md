# ADR-0012: Documentation and Navigation Hardening

## Status
Accepted

## Kontext
Nach mehreren technischen Tranches ist die Struktur robuster geworden, aber der Einstieg für Entwicklung, Betrieb, Handoff und Release muss genauso eindeutig sein wie die Codepfade selbst.

## Entscheidung
Wir führen eine zentrale Dokumentations-Navigation ein und definieren README, Quickstart, Architekturübersicht und Handoff Guide als kanonische Einstiegspunkte.

## Konsequenzen
- weniger Suchaufwand
- klarere Übergaben
- geringere operative Drift
- neue Workflow-/Handoff-Pfade müssen künftig in der Dokumentations-Navigation verankert werden
